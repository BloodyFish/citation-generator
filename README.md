# citation-generator
A python program that generates citations from a URL\
<br>
In the terminal, type `python main.py <URL>`

## Example
If you wanted to cite this article from [The Gaurdian](https://www.theguardian.com/artanddesign/2025/dec/31/picasso-portrait-charity-auction-alzheimers), you would type the following into your terminal:\
<br>
`python main.py https://www.theguardian.com/artanddesign/2025/dec/31/picasso-portrait-charity-auction-alzheimers`\
<br>
This will return the following:
```
Title:  €1M Picasso Portrait Up For Grabs For €100 In Charity Raffle
Date:  ['2025', 'Dec.', '31']
Site Name:  The Guardian
Publisher:  The Guardian
Author(s):  ['Nadia Khomami']

Khomami, Nadia. "€1M Picasso Portrait Up For Grabs For €100 In Charity Raffle." The Guardian, 31 Dec. 2025, https://www.theguardian.com/artanddesign/2025/dec/31/picasso-portrait-charity-auction-alzheimers.
```
